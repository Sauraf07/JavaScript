# Tech Career Navigator

A full-stack application built to guide students and professionals on their tech career journeys. 

## Tech Stack
- **Frontend**: React 18, Vite, Vanilla CSS (Glassmorphism + Dark Mode UI)
- **Backend**: Node.js, Express.js (`type: "module"`)
- **Database**: SQLite3 (Zero-config relational database)
- **Authentication**: JWT & bcrypt

## Features
- **Dynamic User Authentication**: Secure JWT-based login and registration.
- **Profile Initialization**: Seamless onboarding to configure courses and semesters.
- **Smart Career Mapping**: Select a tech tree (Frontend, Backend, DevOps, Data Science, etc.) and get an automatically generated curriculum.
- **Roadmap & Tasks**: Track your progress dynamically with a database-backed task list.
- **Visuals**: Cutting-edge, dynamic UI built with custom abstract 3D artwork generated by AI. Uses best practices in CSS variables, responsive design, and animations.

## How to Run Locally

### Prerequisites
- Node.js (v18+ recommended)

### Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start Full-stack environment (Backend + Frontend simultaneously):
   ```bash
   npm run dev:all
   ```

3. Open your browser to `http://localhost:5173`. The backend will run on `http://localhost:3000`.
